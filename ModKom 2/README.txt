Im Ordner Java Editor die Datei Anmeldefenster doppelklicken und oben auf Play drücken

Im Anmeldefenster kann man sich nun entweder mit dem Adminzugang anmelden (Benutzername: Admin Passwort: 1234)
oder man kann sich einen neuen Account erstellen, mit dem man sich anschließend anmelden kann. Wenn die Daten 
übereinstimmen und auf Login gedrückt wurde, öffnet sich das Chatfenster. Mit dem Button "Neuer Chat" Kann ein
neuer Chat erstellt werden. Es ist auch schon möglich eine Chatnachricht zu schreiben und ein 
Einstellungsfenster zu öffnen.
